# Clicker Exercise

This is a simple React exercise to practice component props, event handling, and basic state management.

## Description

The app renders two clickable buttons. Each button displays an alert message when clicked and logs the message to the console.

## Features

- Two Clicker components with different messages and button texts
- Alert popup on button click
- Console logging for debugging

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` (or the port shown in the terminal).

## Technologies Used

- React 19
- Vite
- JavaScript

This is a basic exercise while learning React fundamentals.
